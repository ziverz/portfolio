A low-latency teleoperation system that maps real-time Meta Quest 3 controller data to a physical 6 DOF robot arm while being in mixed reality. Utilizing a Raspberry Pi and Python socket communication, the project translates controller coordinates into precise physical movement via inverse kinematics.
